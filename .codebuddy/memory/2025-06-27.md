# 2025-06-27

## JobRadar 项目初始化

根据项目说明书 `这是一个非常有挑战性但也极具实用价值的项目...txt` 完成了 JobRadar 全栈项目的初始化搭建。

### 完成的工作

- **Next.js 14 项目配置**: package.json, tsconfig.json (strict mode), next.config.js, tailwind.config.ts, postcss.config.js
- **Prisma 数据库 Schema**: Job, UserFavorite, AiAnalysisTask, AiAnalysisCache 四张核心表，包含索引设计（aiScore, location, fetchedAt, status+createdAt 复合索引）
- **Zod 校验 Schema**: JobIngestSchema (采集数据校验), JobAnalysisSchema (AI输出四维度评分结构化校验)
- **API 路由**:
  - `/api/health` - 健康检查（检测数据库连接）
  - `/api/jobs` - 岗位列表（支持 location/minScore/recommendation/search 筛选和分页）
  - `/api/jobs/ingest` - 岗位入库（X-Local-Secret 鉴权 + Prisma upsert 去重 + 自动创建 AI 任务）
  - `/api/favorites` - 收藏 CRUD（软删除 + 乐观更新支持）
  - `/api/workers/ai-analyze` - AI 批处理 Worker（Bearer Token 鉴权 + 过期锁释放）
- **AI 评服务**: Fixed-Shot Prompt 模板 + 3 个示例 + JD 智能截断(3000字符) + Zod 结构化输出校验 + JSON 修复回退
- **AI Worker 服务**: JD SHA-256 哈希缓存 + 简单锁机制(防重复消费) + 重试机制(最多3次) + 过期锁释放
- **前端 Dashboard**:
  - Sidebar 导航 (所有岗位/我的收藏/数据分析/刷新)
  - FilterBar (城市/最低分/推荐等级/关键词搜索)
  - JobCard (可展开详情, 心形收藏乐观更新, AI评分徽章, 红旗风险, 四维评分展示)
  - AnalyticsCharts (Recharts: 评分分布柱状图, 推荐等级饼图, 城市分布横向柱状图)
  - Pagination 分页
  - Empty State / Loading State / Error State 处理
- **Chrome Extension (Plasmo 框架)**:
  - Content Script: DOM 提取器 (列表页+详情页兼容), 薪资文本解析器, 消息响应
  - Popup: 服务状态检测, 提取触发, 结果展示, Dashboard 跳转
  - Background: Service Worker, 图标动态切换
  - Sync: 3次重试 + 延迟递增 + 本地密钥管理
- **种子数据**: 8 个不同类型岗位(强烈推荐/可考虑/不推荐) + 2 条收藏
- **部署配置**: vercel.json (每10分钟 Cron), .env.example (完整配置模板)
- **README.md**: 完整项目文档(含合规声明、架构图、快速开始、API端点)

### 技术决策
- 数据库: PostgreSQL + Prisma ORM（AI 生成 CRUD 准确率高）
- AI 缓存: JD SHA-256 哈希去重（降低 Token 成本）
- 队列: 纯数据库驱动的轻量级队列（个人工具无需 Redis/BullMQ）
- 扩展框架: Plasmo（React + HMR 热更新）
- 状态管理: React hooks (useJobs/useToggleFavorite)，无 Redux
